# vantagecto.github.io

Official website for Vantage CTO.

## Getting Started

### Prerequisites

- A modern web browser
- [Git](https://git-scm.com/) for cloning the repository

### Installation

```bash
git clone https://github.com/vantagecto/vantagecto.github.io.git
cd vantagecto.github.io
```

### Local Development

Open `index.html` in your browser or use a local development server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

Then visit [http://localhost:8000](http://localhost:8000).

## Deployment

This site is deployed automatically via [GitHub Pages](https://pages.github.com/). Any changes pushed to the `main` branch will be reflected at:

> **https://vantagecto.github.io**
